# 删除账号

* 手动删除账号，立即生效：用浏览器打开地址：https://my.telegram.org/auth?to=deactivate
  输入手机号码后，在 Telegram 客户端上确认或输入客户端上消息里的code

* 自动删除账号：Telegram 账号长期不登录不使用会被系统自动注销，时长可以改为"1月/3月/6月/12月"。

::: danger
删除账号后您的所有数据将被清除，无法恢复！
:::
